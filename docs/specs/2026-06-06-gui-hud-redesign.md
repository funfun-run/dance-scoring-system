# GUI HUD 运动风重设计

> 日期: 2026-06-06 | 状态: 已确认 | 技术栈: Python + tkinter + ttkbootstrap

---

## 1. 设计系统

### 1.1 风格
**Dark Mode OLED + Cyberpunk HUD** — 深黑背景、霓虹强调、微光效、大号数字。

### 1.2 色板（来源：ui-ux-pro-max Fitness/Gym App）

| Token | 色值 | 用途 |
|-------|------|------|
| `bg` | `#0F172A` | 主背景 |
| `card` | `#1E293B` | 卡片/面板 |
| `accent` | `#F97316` | 橙色强调 — 主要操作、跟练边框 |
| `success` | `#22C55E` | 通过绿 |
| `warning` | `#F5A623` | 纠正/警告黄 |
| `danger` | `#EF4444` | 失败红 |
| `text` | `#F8FAFC` | 主文字 |
| `muted` | `#94A3B8` | 次要文字 |
| `border` | `rgba(255,255,255,0.08)` | 卡片边框 |

### 1.3 字体
**Space Grotesk**（来源：ui-ux-pro-max Kinetic Brutalism）
- 标题：700 Bold 18pt
- 得分：700 Bold 72pt（跟练）/ 48pt（结果）
- 正文：400 Regular 11pt
- 等宽：Courier 10pt（文件信息）

### 1.4 效果
- 卡片 hover: 边框变亮 + scale(1.02) 0.2s ease
- 跟练卡片: 橙色 `#F97316` 1px 边框 + 外发光 `box-shadow`
- 段切换: 0.5s 淡入淡出 + "▶ 第 X 段" 大字闪
- 视频导入区: 虚线边框 hover 发光

## 2. 界面总览

单窗口架构（1100×680），底部工具栏常驻，内容区面板切换。仅跟练为独立全屏窗口。

### 界面清单

| # | 界面 | 展示方式 |
|---|------|---------|
| 1 | **主 Hub** — 2×2 卡片 + 工具栏 | 主窗口默认 |
| 2 | **🎬 实时跟练** — 双画面 HUD | 独立全屏窗口 |
| 3 | **📊 离线评分** — 导入 + 参数 + 评分 + 结果 | Hub 内切换 |
| 4 | **📁 练习回顾** — 段列表 + 播放 + 纠正 | Hub 内切换 |
| 5 | **✂️ 视频分割** — 选择 + BPM + 结果 | Hub 内切换 |
| 6 | **工具栏面板** — 设置/NPU/模型/性能 | Hub 内切换 |

### 导航规则
- 点击卡片 → 内容区原地切换，顶部 "← 返回" 按钮
- 底部工具栏 → 同样切换面板，保留返回路径
- 点击 "← 返回" → 回到 Hub 卡片
- 仅 "🎬 跟练" 弹出独立全屏窗口

### 3.4 📁 练习回顾

- 左侧段列表（分数+绿/红左侧色条），点击切换段
- 右侧视频播放器 + 播放控制（⏮ ▶ ⏭）

### 3.5 ✂️ 视频分割

- 视频选择（同拖放+最近列表组件）
- BPM 输入
- 开始分割按钮 → 进度 → 结果（段数 + 每段时间 + 输出目录）
- 打开目录按钮

---

### 3.6 工具栏面板

4 个面板共用一个区域，工具栏按钮切换：
- **⚙️ 设置**：推理后端、摄像头、对齐算法、BPM、阈值、窗口大小
- **🖥️ NPU**：设备名、型号、温度、驱动版本、固件、推理延迟
- **🔧 模型**：当前状态、IR 产物信息、精度选择、重新转换按钮
- **📈 性能**：测试视频选择、帧数、预热轮数、开始测试、结果对比表

## 4. 共享组件

### 4.1 视频导入组件 (`VideoImporter`)

一个可复用组件，包含：
- **拖放区**：120px 高虚线边框区，未导入时显示图标 + 提示
- **已导入态**：缩略图 + 文件名 + 帧数 + 时长 + 重新选择按钮
- **最近列表**：从 `videos/` 和 `~/Videos` 扫描 .mp4/.avi/.mov，最多 5 条，点击选中
- **浏览按钮**：`filedialog.askopenfilename()` 兜底
- 选中状态用绿色勾标识

### 4.2 得分显示器 (`ScoreDisplay`)

- 大号数字（48pt/72pt Space Grotesk Bold）
- 颜色自动按阈值切换（≥60 绿色，<60 红色）
- 可选显示判定标签（合格 ✓ / 不合格 ✗）

### 4.3 段得分条 (`SegmentBar`)

- 水平条，按分数比例填充
- 绿色 ≥ 60，红色 < 60
- 点击可跳转到该段回顾
